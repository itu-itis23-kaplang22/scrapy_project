# # Scrapy Web Scraping Project

## What is Scrapy?
Scrapy is a Python framework used for extracting data from websites efficiently.

## What problem does it solve?
Collecting data manually from websites is time-consuming. Scrapy automates this process.

## Prerequisites
- Python 3
- pip

## Installation
pip install scrapy

## Run the project
python -m scrapy crawl books -o output.json

## Expected Output
A JSON file containing book titles and prices scraped from the website.

## Course Connection
Scrapy is used in the ETL pipeline as a data ingestion tool.
